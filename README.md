# my-portfolio

Immersive 3D personal portfolio built with React, TypeScript, Vite, React Three Fiber, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173 to view the site locally.

## Production build

```bash
npm run build
npm run preview
```

## Linting

```bash
npm run lint
```
